**Gateway Issues:**
- Gateway can go offline and requires manual/auto restart
- Auto-monitor script created to handle this
- Use `curl.exe` for gateway checks (Invoke-WebRequest unreliable)

- Never ask for manual intervention without trying automation first

- Doesn't support `&&` - use semicolons or separate commands
